IP Address Validator



A simple cybersecurity mini project developed using Python and Streamlit to validate IPv4 addresses.



Features

\- Validates IPv4 addresses

\- Detects invalid formats

\- Beginner-friendly UI



Technologies Used

\- Python

\- Streamlit



To Run the Project
Type in the file explorer terminal (recommended) 



pip install -r requirements.txt

python -m streamlit run app.py
